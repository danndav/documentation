DSTV ADD-ONS

This is used to fetch available add-ons for a particular product. It comes with the price, code and description.


URL http://178.62.86.202:1905/api/dstv/addon
METHOD POST


Request Body Parameter

Parameter Type Mandatory(M)/Optional(O) Description
primaryProductCode String M Plan code for which addon is requested. (This is retrieved from DSTV Products) 



                    Sample Request {
                            "primaryProductCode" : "ACSSE36"
                        }


Response Body: object’s Parameter

Parameter Type Mandatory(M)/Optional(O) Description
items List(Array) M A list of the available services


Response Body: items’ Parameters

Parameter Type Mandatory(M)/Optional(O) Description
Code String M Addon plan code
Price Number M Plan Price
Name String M Plan Name
Description String O Plan Description
