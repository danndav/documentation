DSTV CARD VALIDATION

This is used to check the validity of the smart-card number and also in turn used to fetch the customer number for subscription.


URL http://178.62.86.202:1905/api/dstv/inquiry
METHOD POST


Request Body Parameter

Parameter Type Mandatory(M)/Optional(O) Description
primaryProductCode String M Smart-card number to be validated 



                    Sample Request {
                        "number":"4258473222"
                        }


Response object’s Parameter

Parameter Type Mandatory(M)/Optional(O) Description
accountStatus String M Customer’s Account Status
firstName Number M Customer First Name
lastName String M Customer’s Last Name
customerType String M Customer’s Type
code String O Error Code
invoicePeriod String M Invoice period for which the payment will be made
dueDate String M Due Date
errorId String O Error code Id
customerNumber String M Customer’s Number
Message String O Error Message
